# .dotfiles - manage dotfiles

For future use,

- Preview changes after stow: `stow -nvSt ~ <FOLDER>`
- Apply changes: `stow -vSt ~ <FOLDER>`
- Delete stow: `stow -vDt ~ <FOLDER>`
- Force stow for the first time: `stow --adopt -vSt ~ <FOLDER>`

**Notes**:

- Add `-n` flag if you want to preview changes.
- Replace **FOLDER** with the correct folder name you want to apply.
